## 🚀 Featured Projects



### 🎬 [Quiro](https://github.com/Nweremizu/quiro) — Desktop Screen Recorder & Editor
*05/2026 – Present*

A **cross-platform desktop screen recording and editing application** — record, edit, and export professional-quality demos and tutorials in one unified workflow.

**Stack:** `Electron` `React` `TypeScript` `PixiJS` `GSAP` `FFmpeg` `Whisper`

**What it demonstrates:**
- 🖥️ Electron desktop app with native screen capture and system audio recording
- 🎨 PixiJS + GSAP compositing engine for cursor effects, zoom regions, webcam overlays, and keyframe animations
- 📝 Whisper-powered automatic caption generation
- 🚀 FFmpeg-backed MP4/GIF export with CUDA-accelerated GPU rendering

---

### ⛪ [Ekklesia](https://github.com/Nweremizu/ekklesia) — Church Management Platform
*04/2026 – Present*

A **full-stack church operations platform** for membership management, attendance tracking, group coordination, and team administration — built for African congregations.

**Stack:** `Laravel 13` `PHP 8.4` `React 19` `TypeScript` `Inertia.js` `Tailwind CSS v4` `PostgreSQL`

**What it demonstrates:**
- 🏗️ Full-stack Laravel + React SPA via Inertia.js with modular route architecture
- 🔐 Role-based access control with invitation-based onboarding and configurable settings
- 📊 Real-time analytics dashboard (Recharts) for membership trends, attendance, and at-risk member alerts

---

### 📡 [Dispatch-V](https://github.com/Nweremizu/dispatch-v) — Real-Time Geospatial Routing Engine
*10/2025 – 01/2026*

A **high-concurrency dispatch system** achieving <150ms driver-matching latency for 10,000+ simulated concurrent drivers.

**Stack:** `Node.js (Fastify)` `Redis Geo` `PostGIS` `NGINX` `BullMQ` `Prometheus` `Grafana`

**What it demonstrates:**
- ⚡ Redis Geo for hot-path spatial indexing + PostGIS for durable geospatial storage
- 🔄 Event-driven microservice-style infrastructure with WebSocket-aware NGINX reverse proxy
- 📈 Production-grade observability stack validating stability under 500+ req/s
